## Movie Search App

A React-based movie search application using the OMDb API, with search, filtering, favorites, and detailed movie views.

### Features
- Search movies by title
- Filter by type (movie, series)
- Paginated results from the OMDb API
- Movie detail page with info like Year, Genre, Actors, Plot
- Add/remove movies to favorites
- Responsive UI using Tailwind CSS

### Tech Stack
- React
- React Router
- Tailwind CSS
- OMDB API

### Setup
1. npm install
2. npm start

### Installation & Setup

1. Clone the repository:

   git clone <repository-url>
   cd <project-folder>


2. Install dependencies:

   npm install


3. Start the development server:

    npm start


4. Open http://localhost:3000
 in your browser.

### Notes

- Ensure your OMDb API key is set in api.js:

   * const API_KEY = "your_api_key";
   * const API_URL = "https://www.omdbapi.com/";
